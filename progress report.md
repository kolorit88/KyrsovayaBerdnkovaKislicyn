# Бэкенд проекта

### Архитектура
Проект разрабатывается согласно паттерну **«чистая архитектура»**.  
Созданы следующие слои:

- `domain`
- `application`
- `infrastructure`
- `presentation`

### Модели и база данных
- Созданы все необходимые доменные модели.
- Разработана схема базы данных.
- Организовано взаимодействие с БД с помощью **SQLAlchemy** (ORM).
- Выполнена первая ревизия миграции (**Alembic**).
- Развёрнут **PostgreSQL** (пока без Docker).

### DTO и эндпоинты
- Созданы DTO.
- Начата работа над `merchant` router:
  - поддерживаются несколько эндпоинтов.
- Обновляется **OpenAPI**-документ (Swagger), что упрощает использование бэкенда.

### Конфигурация и окружение
- Конфиги (включая пароли) вынесены в отдельные файлы.
- Создан `requirements.txt` для установки зависимостей.
- Бэкенд развёрнут для тестов в локальной сети.

### Планы доработки
Остаётся добавить:

- эндпоинты:
  - авторизации
  - регистрации
  - админ-панели
  - оператор-панели
- внедрить:
  - **JWT**
  - **Refresh Token**

# Дизайн

- Разработаны **все необходимые экраны** для переноса в вёрстку.
- Готовый дизайн-проект присутствует в корне проекта.
